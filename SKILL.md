# JAV 下载工具

JAV 番号查询与下载工具，支持三种模式：

## 功能模式

### 1. 解密隐藏番号
从用户发送的叙事文本中解密嵌入的番号（包含嵌入式字母/数字组合、地名+数字、数据+数字、路线指引+数字等模式）。

### 2. 查询信息
查询片名、女优、封面等信息。

**触发方式：**
- 用户说"XXX-123的片名/女优是什么"
- 用户说"告诉我 XXX-123"

### 3. BT 下载
自动搜索 + 筛选 + 下载 + 清理。

**触发方式：**
- 用户说"下载 XXX-123"
- 用户说"/jav-dl XXX-123"

## 核心原则

- 无码 > 中文 > 种子数
- .torrent 文件 > magnet 磁力链接

## 下载目录

通过 `$JAV_DL_DIR` 环境变量配置，默认为 `$HOME/Downloads/av`。

## AI 模型配置

**推荐使用 DeepSeek 官方 API**，将 Claude Code 的 API 切换至 DeepSeek 官方即可。
